Here is a table that maps each programming language to a wildcard expression for source files:

| Programming language | Wildcard Expression |
|----------------------|---------------------|
| Java                 | *.java              |
| Kotlin               | *.kt                |
| Javascript           | *.js                |
| Typescript           | *.ts                |
| C++                  | *.(h|hpp|cpp|cc)    |
| Gradle Kotlin DSL    | *.kts               |

The following table also maps programming languages to compiled object code wildcards:

| Programming language | Object code wildcard Expression    |
|----------------------|------------------------------------|
| Java                 | *.class                            |
| Kotlin               | *.class                            |
| C++                  | *.(o|obj)                          |
